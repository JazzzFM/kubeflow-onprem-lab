## Demo 2: LeNet-5 CNN (Deep Learning + GPU)

| Métrica | Valor |
|---|---|
| Framework | PyTorch 2.11.0 + CUDA 13.0 |
| Device | **NVIDIA GeForce RTX 5080 Laptop GPU** |
| Compute capability | **sm_120 (Blackwell)** |
| Dataset | MNIST (60k train, 10k test) |
| Parámetros | 61,706 |
| Optimizer / lr / batch | Adam / 1e-3 / 128 |
| **Final accuracy** | **0.986** |
| **Total training time** | **11.73 s (3 epochs)** |
| GPU memory peak | 28.9 MiB |

| Epoch | Loss | Test acc | Tiempo |
|---|---|---|---|
| 1 | 0.3116 | 0.9725 | 7.94 s |
| 2 | 0.0799 | 0.9800 | 2.26 s |
| 3 | 0.0549 | 0.9860 | 1.53 s |

> Epoch 1 tarda ~5x más que epoch 3 por el JIT compile inicial de cuDNN para
> Blackwell sm_120. Es comportamiento normal y conviene mostrarlo a los alumnos.
